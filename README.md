# 🐺 Lobo Sabio TTS - Generador de Voz IA en Español

Una aplicación web local hecha con **Flask** y **Coqui TTS** que convierte texto en voz realista en español. Ideal para crear narraciones, voces para videos o convertir a tu lobo interior en un sabio filósofo parlante.

---

## 🚀 Características

- 🎙️ Conversión de texto a voz en español con IA.
- 🌐 Interfaz web local.
- 📥 Descarga de archivos de audio generados.
- 🐺 Basado en modelos libres y de código abierto.

---

## 🛠️ Requisitos

- Python 3.10+
- Git
- Navegador web moderno (para la interfaz)
- (Opcional) `ffmpeg` para convertir `.wav` a `.mp3`

---
